# Student Mental Health Risk Prediction using Machine Learning

## Project Overview
This project predicts student mental health risk levels using Machine Learning algorithms. The system analyzes student-related data and provides predictions that can help identify students who may require support.

## Features
- Data preprocessing
- Exploratory Data Analysis
- Machine Learning model training
- Mental health risk prediction
- Performance evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Structure
├── dataset.csv
├── app.py

├── requirements.txt
└── README.md
